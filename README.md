# 慧生活798积分获取脚本

## 📝 简介

这是一个用于慧生活798平台的自动化积分获取脚本，支持以下功能：
- ✅ 每日自动签到
- 📺 观看视频获取积分
- 👥 支持青龙面板多账号运行

## 🔧 环境要求

- Python 3.6+
- requests 库

安装依赖：
```bash
pip install requests
```

## ⚙️ 环境变量配置

### HUI798_AUTH

**必填** - 授权令牌（Authorization）

#### 获取方式

1. 打开慧生活798 APP 或小程序慧生活798
2. 登录账号后，使用抓包工具或浏览器开发者工具
3. 访问任意 API 接口，例如：`https://i.ilife798.com/api/v1/ui/app/master`
4. 在请求头中找到 `authorization` 字段，复制其值即为 `HUI798_AUTH`

#### 设置方法

**单个账号：**
```bash
export HUI798_AUTH='你的授权令牌'
```

**多个账号：**
支持多种分隔符：`&`、`#`、`@`、`|`、`,`

```bash
export HUI798_AUTH='token1&token2&token3'
# 或
export HUI798_AUTH='token1#token2#token3'
# 或
export HUI798_AUTH='token1@token2@token3'
```

## 🚀 使用方法

### 本地运行

```bash
# 设置环境变量
export HUI798_AUTH='你的授权令牌'

# 运行脚本
python iLife798.py
```

### 青龙面板

1. 将 `iLife798.py` 上传到青龙面板的 scripts 目录
2. 在青龙面板 **环境变量** 中添加：
   - 名称：`HUI798_AUTH`
   - 值：你的授权令牌（多账号用分隔符隔开）
3. 添加定时任务：
   ```
   0 8 * * * python3 /ql/scripts/iLife798.py
   ```

## 📊 功能说明

| 功能 | 说明 | 积分奖励 |
|------|------|----------|
| 每日签到 | 每天签到一次 | 5积分 |
| 观看视频 | 每次执行观看5次视频 | 每次30积分 |

## 📋 运行流程

1. 从环境变量读取授权令牌
2. 获取任务列表并显示用户信息
3. 执行每日签到
4. 等待10秒后开始观看视频（共5次）
5. 多账号模式下，账号间隔15秒执行

## ⚠️ 注意事项

- 授权令牌有有效期，过期后需要重新获取
- 如果提示"登录状态已过期"，请更新 `HUI798_AUTH`
- 请求过于频繁时脚本会自动等待5秒后重试
- 建议每天只运行一次，避免频繁请求
